# ServerOps Monitor

Telegram bot for monitoring and managing servers via Ansible.

## Features
- Add servers via Telegram
- Store server data in PostgreSQL
- Run checks using Ansible
- View server list and status

## Tech Stack
- Python (aiogram)
- PostgreSQL (SQLAlchemy)
- Python linter (Ruff)
- Ansible


## Usage
1. Start bot with /start
2. Add server (IP, login, password)
3. Run check
